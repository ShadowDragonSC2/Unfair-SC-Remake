# FIRE WALL

Модификация добавляет на карту стены огня со всех 4-рех сторон, которые движутся в противоположенную сторону по своему направлению

- **Левая** стена движется направо
- **правая** на лево
- **верхняя** движется вниз
- **нижняя** движется вверх
